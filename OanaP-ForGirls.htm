Girls who Code

Girls Who Code programs work to inspire, educate, and equip girls with the computing skills to pursue 21st century opportunities.

Black girls code

Black Girls CODE introduces computer coding lessons to young girls from under-represented communities in programming languages such as Scratch or Ruby on Rails. Black Girls CODE has set out to prove to the world that girls of every color have the skills to become the programmers of tomorrow.

Made with code

Code can be the most creative tool in your toolbox. Meet the full-time dreamers, beautiful-makers, and culture hackers whose code is changing the world.
